# IWantU
## 文件说明
```
black.txt     黑名单 包含黑名单中的元素的地址都会被过滤
ip.txt        IP列表 数据筛选的对象
keywords.txt  关键字 包含关键字中一个或多个的地址都会被筛选
white.txt     白名单 必须包含白名单中出现的每一个元素
```

## 运行环境
> python 3.7

## 需要安装的库 xlwt
```
pip3 install xlwt
```
